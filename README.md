# بوت تيليجرام للتحميل — QoranSnaptubeBot

بوت تيليجرام مكتوب بـ Python يدعم تحميل المحتوى من يوتيوب وتيك توك وإنستقرام وبينترست، مع نظام تحكم خاص بالمطور مرتبط بـ Supabase.

## الميزات

### القسم العام
- `/start` — رسالة ترحيبية
- إرسال أي رابط → **3 أزرار**: 🎥 فيديو | 🎵 صوت | 🎥🎵 كلاهما
- يوتيوب: فيديو حتى 1080p أو MP3
- تيك توك: بدون علامة مائية
- إنستقرام: ريلز وفيديوهات وصور (مع إصلاح مشكلة الشاشة السوداء)
- بينترست: فيديوهات وصور
- حد 50 ميجابايت مع رسائل واضحة عند التجاوز

### قسم المطور
- `/setContent_chroma` — نشر فيديو في قناة الكروما
- `/setContent_nature` — نشر فيديو في قناة المناظر الطبيعية
- يحفظ البيانات تلقائياً في Supabase

## البنية
```
main.py              — نقطة الدخول
config.py            — الإعدادات والثوابت
utils.py             — دوال مساعدة
fetchers.py          — جلب بيانات المنصات
media_tools.py       — أدوات ffmpeg وyt-dlp
handlers/
  commands.py        — أمر /start
  url_handler.py     — معالج الروابط والأزرار
  downloaders.py     — منطق التحميل لكل منصة
  admin.py           — أوامر المطور + Supabase
supabase_setup.sql   — سكريبت إنشاء جدول قاعدة البيانات
```

## المتغيرات البيئية
```
BOT_TOKEN    — توكن بوت تيليجرام (مطلوب)
```

## التشغيل
```bash
pip install -r requirements.txt
python main.py
```

## قاعدة البيانات
شغّل `supabase_setup.sql` في Supabase Dashboard → SQL Editor لإنشاء الجدول والسياسات.
